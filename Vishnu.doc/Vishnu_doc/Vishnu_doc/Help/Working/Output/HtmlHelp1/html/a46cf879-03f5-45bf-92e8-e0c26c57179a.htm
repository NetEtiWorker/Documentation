Version 7.0.1.6

Version 7.0.1.6 Hard Core (Stand: 16.10.2023)

Dieser Version ist die erste komplett auf .Net 7.0 migrierte Vishnu-Software.
Dies umfasst auch das Framework, alle Plugins und die Dokumentation. Diese Version ist getestet und stabil.

Änderungen in dieser Version

  • Der Vishnu-Installer (ClickOnce) wurde nach Migration auf .Net 7 deaktiviert. Hier muss langfristig eine bessere Lösung gefunden werden.

  • Die VSIX-Templates zur Unterstützung der Erstellung eigener Vishnu-Checker und User-Controls wurden zwar nach Migration auf .Net 7 wieder funktionstüchtig gemacht, erzeugen aber bei der Generierung neuer Projekte noch Fehlermeldungen. Ansonsten funktioniert aber alles.

  • Erstellung von NuGet-Packages über eine allgemeingültige Lösung für alle Projekte über Environment-Variable NETETI_CIB=true und ...\AddOn\proj\Conditional_NuGet_Package_Properties.proj implementiert.

  • MicroMailer (und alle sonstigen Worker): Parameter überarbeitet und, wo nötig, ausgelagert.

  • Vishnu-Positionierung und Resizing bei Multiscreen-Orientierung (-SizeOnVirtualScreen=true) überarbeitet.

  • Die Ausrichtung des Vishnu-Trees kann jetzt zur Laufzeit zusätzlich zum Klick auf das YinYang-Symbol auch über F4 geändert werden.

  • Vishnu-eigene Meldungen mit korrekter Positionierung, Zoom-Funktionen und Icons implementiert.

  • ToolTips mit Copy-Funktion realisiert.

  • InPlug/WebScraper: eigene Lösung über Verkürzen und Herunterzählen der Chrome-Version für den Download einer passenden Treiberversion implementiert.
