TreeEventTrigger Eigenschaften

Der TreeEventTrigger Typ exponiert die folgenden Member.

Eigenschaften
 NameBeschreibung
Info Enthält Informationen zum besitzenden Trigger. Implementiert sind NextRun und NextRunInfo. Für das Hinzufügen weiterer Informationen kann diese Klasse abgeleitet werden.
InternalEvents Ein normalisierter String mit durch '|' getrennten internen Event-Namen.
IsActive Ersetzt das Starten und Stoppen bei internen Triggern.
LastTreeEvent Das letzte aufgetretene TreeEvent für diesen Trigger oder null.
OwnerId Id des Knotens, der von dem Event getriggert wird (nicht des Knotens, der das Event auslöst).
ReferencedNodeId Enthält den Namen des referenzierten Originalknotens.
