CustomProgressBar Eigenschaften

Der CustomProgressBar Typ exponiert die folgenden Member.

Eigenschaften
 NameBeschreibung
IsGlowing Schaltet den Glow-Effekt der ProgressBar an oder aus.
IsIndeterminate Schaltet zwischen von links nach rechts laufender und ewig hin und her laufender ProgressBar um.
Text Setzt oder liest einen auf der ProgressBar zentrierten Zusatztext.
Value Prozentwert der ProgressBar - kann von außen gelesen und gesetzt werden.
