Vishnu.WPF_UI.ValueConverter Namensraum

Hilfsklassen für die WPF-Bildschirmoberfläche von Vishnu
Klassen
 KlasseBeschreibung
ChildOrientationToExpandDirection ValueConverter, wandelt Die Child-Anordnung eines Knotens (Horizontal, Vertical) in eine ExpandDirection für den anzeigenden Expander (Down, Right).
EqualityConverter Compares two Properties (Thanks to Jason Tyler).
NotEmptyToTrue ValueConverter, wandelt einen nicht leeren String in Boolean True.
NullableBoolToBrush ValueConverter, wandelt einen nullable bool in Farben: null: gelb, true: grün, false: rot.
OrientationModifier ValueConverter, wandelt eine Orientation (Horizontal, Vertical) abhängig von einem zusätzlichen Schalter vom Typ OrientationSwitch um: Unchanged=unverändert, Switched=umgedreht, Horizontal=horizontal, Vertical=vertikal.
Path2UserControlBase ValueConverter, wandelt einen nicht leeren String in Boolean True.
VisualNodeStateToBitmapImage Setzt einen Enum-Typ in ein BitmapImage um.
VisualNodeWorkerStateToText ValueConverter, wandelt einen nullable bool in Farben: null: gelb, true: grün, false: rot.
